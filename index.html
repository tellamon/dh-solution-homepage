Applications

PC 환경은 사용자 친화적 UX, 연결성, 고속처리 등 비즈니스 솔루션에서 가장 큰 범용성을 가진 분야입니다. .NET, Qt와 같은 크로스 플랫폼 프레임워크를 활용하여 Windows, Linux, Mac OS X에서 구동 가능한 데스크톱 애플리케이션을 제작합니다.

Android, iOS 모바일 환경에서도 Ionic, Xamarin 등을 활용하여 앱 제작부터 마켓 배포와 업데이트 관리까지 처리합니다.

.NET Framework, DevExpress, Qt, MSSQL DataBase, Xamarin, Ionic, Electron, React Native

Web Services

웹은 모든 기기와 플랫폼에서 최종 사용자와 만나는 가장 친숙한 접점입니다. 디에이치솔루션은 브라우저에 표시되는 프론트엔드부터 서버와 연결되는 서비스까지 전부 개발합니다.

단일 페이지 앱은 자연스러운 UX와 빠른 성능을 제공하고, 웹 애플리케이션 서버는 REST API 형태로 다양한 요구사항을 처리합니다. OAuth2.0, OIDC, 블록체인 노드 연동, 미디어 처리까지 폭넓게 대응합니다.

Typescript, Angular, React, ASP.NET Core, Entity Framework, IdentityServer, Azure SQL, Truffle, OpenZeppelin

Embedded

저렴해진 임베디드 시스템 보드를 활용하여 공장자동화부터 일상 생활 기기에 이르는 다양한 솔루션을 개발합니다. OS 포팅과 애플리케이션 개발을 통해 센서 입력을 처리하고, 디스플레이 또는 네트워크를 통해 서버와 통신하는 IoT 환경을 구축합니다.

고성능 ARM 보드에서 안드로이드를 포팅하여 영상 플레이백과 스트리밍을 제공하는 TV 박스 형태의 제품도 구현합니다.

Raspberry Pi, Arduino, FFMpeg

Machine Intelligence

풍부한 데이터, 컴퓨팅 성능 향상, 오픈소스 라이브러리로 인공지능은 4차 산업혁명의 중심에 서게 되었습니다. 디에이치솔루션은 저수준 영상처리부터 딥러닝까지 컴퓨터 비전 중심으로 연구합니다.

CNN, RNN을 활용한 분류, 검출, 영역분할 등 고수준 처리를 통해 이미지와 영상에서 의미 있는 데이터를 추출합니다.

FFMpeg, OpenCV, PyTorch, TensorFlow, Azure AI
