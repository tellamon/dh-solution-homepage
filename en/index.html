Applications

PC application is the most flexible choice for complex business requirements because it gives user-friendly UX, stable connectivity, and faster processing power. We utilize frameworks such as .NET and Qt to develop software that runs on Windows, Linux, and Mac OS X.

We also make Android and iOS mobile apps with cross-platform frameworks such as Ionic and Xamarin, then deploy and manage updates through app markets.

.NET Framework, DevExpress, Qt, MSSQL DataBase, Xamarin, Ionic, Electron, React Native

Web Services

The web has become the preferred UX endpoint across mobile and desktop platforms. DH Solution develops full-stack web services from front-end clients to server-side APIs.

SPA clients provide natural and fast redraw experiences, while web application servers provide REST APIs, database operations, OAuth 2.0 and OIDC services, blockchain node transactions, and media processing.

Typescript, Angular, React, ASP.NET Core, Entity Framework, IdentityServer, Azure SQL, Truffle, OpenZeppelin

Embedded

Using affordable embedded systems, we develop solutions from daily appliances to factory automation. We tweak OS builds, develop applications, and process inputs from accelerometers, thermometers, light, infrared, and ultrasonic sensors.

Processed data can be displayed directly or sent to servers through Bluetooth or Wi-Fi connected environments.

Raspberry Pi, Arduino, FFMpeg

Machine Intelligence

Abundant data, high performance computing power, and open source libraries make artificial intelligence a key factor for the fourth industrial revolution. DH Solution researches computer vision from basic image processing to deep learning.

Using models such as CNN and RNN, we extract meaningful information from images and videos through classification, detection, and segmentation.

FFMpeg, OpenCV, PyTorch, TensorFlow, Azure AI
