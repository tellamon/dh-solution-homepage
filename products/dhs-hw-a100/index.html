제품사양

Packaging, Features, Isolation, Relay, Power Module

Packaging

  • Raspberry PI 4B 2G x 1
  • MLC Micro SD card 32G x 1
  • 220v Power cord 1.5m x 1

Features

  • Size: 134mm (W) x 107mm (L) x 73mm (H)
  • Operating current: 5mA Idle, 100mA Working
  • Operating voltage: 5VDC
  • Operating Temperature: -30 ~ +70°C
  • Operating Humidity: 20 to 85% RH
  • External Raspberry PI 4B Mount (GPIO 3.3vdc)

4Ch Photocoupler Isolation

  • Signal Isolation Input and Output level converting
  • Operating voltage: Input 24VDC / Output 3.3VDC
  • 80kHz Conversion frequency
  • Reverse-polarity protection
  • Optional common ground

1Ch Relay

  • Arrangement: 1 From C, 10A
  • From A: 10A/125VAC, 5A/250VAC, 5A/30VDC
  • From B: 3A/125VAC, 2A/250VAC, 1A/30VDC
  • Mechanical life: min. 10 million / at 180 times/min
  • Operate: Max 20ms, Release: Max 20ms

Power Module

  • Universal AC Input (85 ~ 264VAC) / Full range
  • Efficiency: 79%
  • Protections: short circuit, overload, over voltage
  • Isolation class II
  • Output DC Voltage: 5VDC
  • Rated current: 4A/20W

Application Pinouts

This device has external pull-down registers for signal input ports. There is no need to use RPI internal pull-up/pull-down register. If you want to use the RPI internal registers, double-check the hardware specifications of both RPI and DHS-HW-A100-PRC.
